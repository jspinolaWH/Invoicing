| # | Criterion | Status | Notes |
|---|-----------|--------|-------|

| 2 | System applies all invoicing rules (billing cycles, pricing, VAT, allocations) | IMPLEMENTED | `InvoiceGenerationService.generate(..., true)` runs full pipeline in simulation mode |

| 9 | If errors detected, system flags affected invoices for correction | IMPLEMENTED | BLOCKING failures disable "Commit Real Run" button in `SimulationResultsPage` |
