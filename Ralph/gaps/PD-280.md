## Gaps Found

### Gap 1: Missing database migrations for property_groups, shared_service_participants tables and shared_service_group_id column
- **Layer**: Backend
- **Criterion**: AC 9 (and transitively all ACs since the feature cannot run without the tables)
- **What exists**: PropertyGroup.java and SharedServiceParticipant.java JPA entities targeting tables `property_groups` and `shared_service_participants`. BillingEvent.java has a `shared_service_group_id` column field. spring.jpa.hibernate.ddl-auto=none so Hibernate will NOT auto-create them. Migrations V1–V27 exist; none create these tables or the billing_events column.
- **What is missing**: A Flyway migration file (V28) creating: (1) `property_groups` table, (2) `shared_service_participants` table with FK to property_groups, and (3) `ALTER TABLE billing_events ADD COLUMN IF NOT EXISTS shared_service_group_id VARCHAR(100)`.
- **Files to change**: Create `invoicing/src/main/resources/db/migration/V28__property_groups.sql`
