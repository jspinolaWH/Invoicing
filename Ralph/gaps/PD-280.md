## Gaps Found

### Gap 1: Missing database migration for property_groups and shared_service_participants tables
- **Layer**: Backend
- **Criterion**: All (without migration the tables don't exist and the application cannot start or process shared services)
- **What exists**: PropertyGroup.java and SharedServiceParticipant.java JPA entities referencing tables `property_groups` and `shared_service_participants`. Migrations V1–V19 exist but there is no V20.
- **What is missing**: A Flyway migration file creating both tables with all required columns matching the JPA entities: `property_groups` (id, name, description, active, created_by, created_at, last_modified_by, last_modified_at) and `shared_service_participants` (id, property_group_id, customer_number, share_percentage, valid_from, valid_to, include_if_zero_share, created_by, created_at, last_modified_by, last_modified_at).
- **Files to change**: `invoicing/src/main/resources/db/migration/V20__shared_services.sql` (new file)
